# This is NOT the Next.js you know
Project Prompt: Atariki Japan Website Development
Role: Senior Full-stack Web Developer & UI/UX Designer.
Objective: Develop a modern, high-conversion landing page for "Atariki Japan," a Japanese restaurant. The website must reflect the brand's bold red aesthetic and premium-yet-accessible food quality.

1. Tech Stack Preferences
Frontend: Next.js (App Router), Tailwind CSS, Framer Motion (untuk animasi).

Icons: Lucide-react atau FontAwesome.

Font: Inter atau Montserrat (untuk tampilan modern) dikombinasikan dengan font Jepang (seperti Noto Sans JP).

2. Visual & Brand Identity (Critical)
Color Palette:

Primary: #C41212 (Bold Red - sesuai gambar).

Secondary: #FFFFFF (White) & #1A1A1A (Deep Black) untuk kontras tinggi.

Accents: Golden Yellow (seperti teks "BEEF CURRY") untuk elemen Call-to-Action (CTA).

Style: Grunge/Textured overlay pada elemen besar, tipografi tebal (bold), dan penggunaan foto makanan yang high-definition dengan drop shadow halus.

3. Website Structure (Sitemap)
Hero Section: Image slider atau video background pendek berkualitas tinggi, Headline besar ("限りない喜び - Endless Joy"), dan tombol "View Menu".

Featured Menu (Grid): Card layout menampilkan menu unggulan (seperti Beef Curry di gambar) dengan harga dan deskripsi singkat.

About Us: Narasi singkat mengenai konsep Atariki Japan (Modern Japanese Fast Casual).

Location/Contact: Integrasi Google Maps (khususnya untuk area operasional saat ini), jam operasional, dan link WhatsApp untuk reservasi/order.

Social Feed: Section "Follow Us on Instagram" yang menampilkan grid foto dari @atarikijapan.

4. Functional Requirements
Responsive Design: Website harus mobile-first karena target pelanggan kemungkinan besar membukanya via smartphone.

Interactive Menu: Filter kategori menu (Curry, Ramen, Sides, Drinks) tanpa reload halaman.

Fast Loading: Optimasi gambar (WebP format) agar performa LCP (Largest Contentful Paint) tetap hijau.
